# floweelyy
Flowers site

## миграции базы данных
```
flask db init
flask db migrate -m "comment"
flask db upgrade
```

## Создаём админа вручную
Перед первым запуском добавим администратора:
```
python add_administrator.py
```

## Запуск
`flask --app main run`
